## React Simple Modal

Simple library with Modal component for React.

## Installation

`npm i react-awesome-simple-modal`

## Demo

## API

| Property | Type       | Description                  |
| :------- | :--------- | :--------------------------- |
| open     | boolean    | Trigger to open/close modal. |
| onClose  | () => void | Handle modal closing action. |
